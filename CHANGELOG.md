# Changelog

All notable changes to Nimdalcraft are tracked here.

## 0.1.0 - 2026-03-24

- Reframed Nimdalcraft from a repo recommender into a code retrieval and reconstruction engine.
- Added feature extraction, retrieval maps, credibility scoring, and reconstruction plan outputs.
- Added the npm CLI entrypoint for setup, doctor, run, and starter validation workflows.
- Added CI jobs for daily GitHub snapshot refresh and validated starter status refresh.
- Added degraded search mode backed by a CI-built GitHub search snapshot.
